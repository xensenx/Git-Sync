What it does

Everything you need, nothing you don't.

🔄

Smart Sync

Three-way diff engine compares local, base, and remote states. Only changed files are touched — fast and bandwidth-friendly.

📱

Mobile-First

Works identically on iOS and Android Obsidian. No workarounds, no companion apps. Pure API calls.

⚡

Auto Sync

Set a quiet idle timer and DGS syncs automatically in the background. Zero manual effort after setup.

🔒

Private Repos

Full support for private repositories via Personal Access Tokens. Your notes stay yours.

🚫

Path Ignoring

Exclude files or folders with simple glob patterns. .obsidian/ is excluded by default.

🏷️

Device Labels

Tag each commit with a device name. Know exactly which machine or phone made a change.

The process

How it works

1

Connect

Enter your GitHub PAT, username, and repo name in the plugin settings. One-time setup.
2

Pull

DGS fetches the remote repo tree and downloads only what's changed since your last sync.
3

Write

Take notes in Obsidian as normal. DGS watches for changes in the background.
4

Push

Tap sync or let auto-sync fire. Changed files are committed and pushed to GitHub instantly.
