Support the Project

Direct GitHub Sync is free, open source, and built by one person in their spare time. Here's how you can help.

Author's Note
S

Hey, I'm Sen.

I built Direct GitHub Sync because I wanted to back up my Obsidian vault across all my devices without installing Git on my phone or paying for a sync service. I figured if I needed it, other people probably did too.

DGS has grown more than I expected, and I'm happy to keep improving it. It's free and will stay free. If it's been useful to you, the best things you can do are: star the repo, share it with other Obsidian users, report bugs, or throw a coffee my way if you're feeling generous.

Privacy Policy

Direct GitHub Sync does not collect, store, or transmit any personal data to third-party servers. All data processed by this plugin stays between your device and GitHub's API using your own credentials.

What the plugin accesses: Your vault files (to sync them), your GitHub Personal Access Token (to authenticate with GitHub's API), and plugin settings stored locally on your device.

What the plugin does NOT do: It does not send data to any server other than GitHub's official API (api.github.com). It does not track usage, collect analytics, or phone home.

Your PAT is stored in Obsidian's local plugin data. Keep it secure — do not share your vault's .obsidian/plugins/ folder or its settings file publicly.
